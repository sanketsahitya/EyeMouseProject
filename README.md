# Mouse Control Using Eye Movement

Mouse Control Using Eye Movement is a computer vision based project that allows a user to control the computer mouse using eye movements detected through a webcam. The system captures video frames, detects eye position, and translates the movement into cursor actions.

This project demonstrates the use of computer vision techniques for human computer interaction.

## Features

Real time eye movement detection  
Mouse cursor control using eye movement  
Webcam based tracking  
Simple and lightweight implementation  

## Technologies Used

Python  
OpenCV  
NumPy  
PyAutoGUI  

## How It Works

The program captures video from the webcam and processes each frame using OpenCV. Eye movements are detected using computer vision techniques, and the detected direction is mapped to mouse cursor movement using PyAutoGUI.

## Requirements

Python 3.x  
OpenCV  
NumPy  
PyAutoGUI  

## Installation

1. Clone the repository

```
git clone https://github.com/sanketsahitya/EyeMouseProject.git
```

2. Install required libraries

```
pip install opencv-python numpy pyautogui
```

3. Run the program

```
python eye_mouse.py
```
